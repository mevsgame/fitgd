# Forged in the Grimdark

A Forged in the Dark tabletop RPG for grim dark science fiction settings.

## What's Inside

- **rules_primer.md** - The complete core rules for the game system, based on Blades in the Dark
- **content/** - Setting-specific content including traits, gear, and examples

## Getting Started

1. Read the **rules_primer.md** for the complete game rules
2. Browse **content/trait.md** for character creation options
3. Check **content/gear.md** for equipment
4. See **content/examples.md** for gameplay examples

Enjoy your adventures in the grim darkness!
